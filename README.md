## Sections

1. **Header** - Name and professional title
2. **About Me** - Short bio
3. **Skills** - Bullet list of technologies
4. **Experience** - Table with position, company, and duration
5. **Projects** - Six real projects with technologies and impact
6. **Education** - Table with degree, institution, and year
7. **Contact** - Email, phone, location, LinkedIn

## Setup

No build step required.

```bash
# Clone the repo
git clone https://github.com/<your-username>/<repo-name>.git
cd <repo-name>

# Open in browser
open index.html          # macOS
xdg-open index.html      # Linux
start index.html         # Windows
```

Or drag `index.html` into any browser.

## HTML Semantics Used

- `<header>`, `<section>`, `<article>`, `<footer>` - structural tags
- `<h1>`, `<h2>`, `<h3>` - heading hierarchy
- `<p>` - paragraphs
- `<ul>`, `<li>` - unordered lists
- `<table>`, `<thead>`, `<tbody>`, `<tr>`, `<th>`, `<td>` - tabular data
- `<hr>` - section dividers
- `<a>` - external links
- `<b>` - inline emphasis for contact labels

## Layout Notes

- Spacing achieved via legacy HTML body attributes (`leftmargin`, `rightmargin`, `topmargin`) - keeps the file **CSS-free** per assignment rules.
- Table borders via HTML `border` attribute.
